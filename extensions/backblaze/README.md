# Backblaze

> A [Vicinae](https://github.com/vicinaehq/vicinae) extension to manage your [Backblaze B2](https://www.backblaze.com/cloud-storage) cloud storage account.

## Demo

![Screenshot](./assets/preview.png)

## Features

- List Backblaze B2 Buckets

## Usage

## Extension Preferences

To use this extension, you need to set up your Backblaze B2 credentials in the extension preferences.

You will need:

- **Account ID**: Your Backblaze B2 Account ID.
- **Application Key**: Your Backblaze B2 Application Key.

You can find these credentials in your Backblaze B2 account under the [App Keys](https://secure.backblaze.com/app_keys.htm) section.

## Development


```bash
npm install
```

To run the extension in development mode, use:

```bash
npm run dev
```

To build the extension for production, use:

```bash
npm run build
```
